# Python - More Data Structures: Set, Dictionary

This project contains Python programs that demonstrate the use of:

- Sets
- Dictionaries
- Lambda functions
- Map, filter, and reduce
- Matrix operations
- Data manipulation

## Learning Objectives

By the end of this project, you should be able to:

- Understand and use sets and dictionaries.
- Manipulate collections efficiently.
- Apply lambda functions with map().
- Solve programming problems using Python data structures.
- Write clean, PEP 8 compliant Python code.

## Requirements

- Python 3
- Ubuntu 20.04 LTS
- All files are executable.
- Code follows pycodestyle (PEP 8).
