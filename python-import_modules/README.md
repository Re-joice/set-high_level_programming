# Python - Import & Modules

This project introduces Python modules and demonstrates how to import functions, variables, and modules in different ways. It also covers the use of command-line arguments and the `__name__ == "__main__"` construct to ensure code is executed only when a file is run directly.

## Learning Objectives

By the end of this project, I will be able to:

- Import functions from another file.
- Use imported functions and variables.
- Create and use Python modules.
- Understand the `__name__` variable.
- Prevent code from executing when a module is imported.
- Use command-line arguments in Python programs.

## Requirements

- Python 3
- Ubuntu 20.04 LTS
- All files start with:

```python
#!/usr/bin/python3
```

- Code follows `pycodestyle`.
- All files are executable.
