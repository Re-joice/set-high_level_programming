# Python - Data Structures

This project contains Python programs that demonstrate the use of lists, tuples, and other basic data structures. The tasks focus on creating, accessing, modifying, and manipulating data structures while following Python best practices and PEP 8 style guidelines.

## Learning Objectives

- Print elements of a list
- Access and modify list items
- Work with tuples
- Use loops with data structures
- Apply Python built-in functions
- Write clean, readable Python code

## Requirements

- Python 3.8.x
- Ubuntu 20.04 LTS
- pycodestyle compliant
- All scripts start with `#!/usr/bin/python3`
