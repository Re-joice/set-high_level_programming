# Python - More Classes and Objects

This directory contains solutions for the **Python - More Classes and Objects** project.

## Learning Objectives

By completing these tasks, I learned how to:

- Create and use classes in Python
- Work with instances and objects
- Use instance attributes and class attributes
- Implement methods and special (magic) methods
- Control attribute access with properties
- Validate attribute values
- Create object representations
- Compare objects
- Understand object-oriented programming (OOP) principles
